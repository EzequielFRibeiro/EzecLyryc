=== COMANDOS PARA GIT ===

git init
git add .
git commit -m "Initial commit: CifraPartit MVP completo - Backend + Frontend + 300 testes"
git remote add origin https://github.com/SEU_USUARIO/cifrapartit-music-transcription.git
git branch -M main
git push -u origin main


=== COMANDOS PARA VERCEL (Frontend) ===

cd frontend
npm i -g vercel
vercel login
vercel --prod


=== COMANDOS PARA RAILWAY (Backend) ===

npm i -g @railway/cli
railway login
cd backend
railway init
railway add --database postgresql
railway add --database redis
railway up


=== VARIÁVEIS DE AMBIENTE - VERCEL ===
